# AETHER Elite SMMA SPA

A premium, production-ready single-page application for an elite Social Media Marketing Agency (SMMA). 

## Design Philosophy
- **Luxury Minimalism**: dark-mode matte palette.
- **Igloo.inc Inspired**: Floating capsule navigation, glassmorphism, and large rounded buffers.
- **Architectural Strategy**: Grid-based layouts with massive typography.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- Bootstrap 5 (Grid & Utilities)
- Lucide Icons
- JavaScript (Vanilla)

## Features
- Glassmorphic Floating Navbar
- Portfolio Architecture Drawer
- Responsive Lead Capture Form
- Strategic Process Timeline
